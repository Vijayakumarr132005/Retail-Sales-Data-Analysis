# Retail-Sales-Data-Analysis
# 🛒 Retail Sales Analytics | End-to-End Data Analytics Project

## 📌 Project Overview

This project demonstrates an end-to-end Data Analytics workflow using **Python**, **SQL**, and **Power BI**. The objective is to transform raw retail sales data into meaningful business insights through data cleaning, SQL analysis, and an interactive Power BI dashboard.

This project showcases practical data analytics skills, including data preprocessing, SQL querying, business intelligence reporting, and dashboard design.

---

## 🎯 Project Objectives

- Clean and preprocess raw retail sales data.
- Perform business analysis using SQL.
- Build an interactive dashboard in Power BI.
- Generate actionable insights to support business decision-making.

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| 🐍 Python (Pandas, NumPy) | Data Cleaning & Preprocessing |
| 🗄️ PostgreSQL | Data Analysis using SQL |

## 📂 Project Workflow

### Step 1 – Data Collection
- Imported the retail sales dataset from Excel.

### Step 2 – Data Cleaning (Python)
- Removed duplicate records
- Handled missing values
- Corrected data types
- Standardized column names
- Exported the cleaned dataset

### Step 3 – Data Analysis (SQL)
Performed SQL queries to analyze:
- Total Revenue
- Total Orders
- Total Customers
- Revenue by Product Category
- Revenue by State
- Monthly Revenue Trend
- Orders by Shipping Method
- Customer Revenue Analysis

### Step 4 – Dashboard Development (Power BI)
Created an interactive dashboard with:
- 📈 Total Revenue KPI
- 📦 Total Orders KPI
- 👥 Total Customers KPI
- ⭐ Average Rating KPI
- 📊 Revenue by Product Category
- 📅 Monthly Revenue Trend
- 🌍 Revenue by State (Map)
- 🍩 Revenue Distribution by State
- 🚚 Orders by Shipping Method
- 🎛 Interactive Slicers
- 📋 Customer Revenue Table
